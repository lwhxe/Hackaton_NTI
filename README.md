# There Before U (TBU)

En sida som är prototyp till ett hackathon projekt under 2025.

---
Användningsområdet gäller sjukvård och allmän blåljuspersonal där det kan vara bra att förutspå var och när saker kan ske.
Projektet är skrivet i (HTML/CSS/JS) och ska i verkligheten använda en rust backend för träningen av AI modellen.
